# Non-Board ChessRTK Logo Concepts

These concepts avoid the chessboard motif. They focus on ChessRTK as a command
line, engine, data, and publishing toolkit.

| File | Concept | Core idea |
| --- | --- | --- |
| `01-cli-cursor.svg` | CLI Cursor | A terminal-native toolkit mark. |
| `02-engine-wave.svg` | Engine Wave | Evaluation signals without board imagery. |
| `03-hash-fingerprint.svg` | Hash Fingerprint | Stable identities for reproducible runs. |
| `04-search-dag.svg` | Search DAG | A compact graph for explored lines. |
| `05-pipeline-arc.svg` | Pipeline Arc | Input, analysis, export, and publishing. |
| `06-dataset-stack.svg` | Dataset Stack | Records layered into training artifacts. |
| `07-pdf-press.svg` | PDF Press | Native publishing as a project identity. |
| `08-protocol-bus.svg` | Protocol Bus | UCI traffic normalized into stable output. |
| `09-repro-seal.svg` | Repro Seal | Deterministic command runs with a checkable result. |
| `10-eval-contour.svg` | Eval Contour | Scores, principal lines, and confidence signals. |
| `11-tensor-cube.svg` | Tensor Cube | Model-ready data without literal chess imagery. |
| `12-config-switchboard.svg` | Config Switchboard | Engines, models, and exports routed by config. |

Open `preview.html` in a browser to compare the full set.
