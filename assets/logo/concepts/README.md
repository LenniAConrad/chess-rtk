# ChessRTK Logo Concepts

These are exploratory vector logo directions for ChessRTK. They are not yet the
canonical project logo.

| File | Idea | Best use |
| --- | --- | --- |
| `perft-path.svg` | Board plus branching legal-move path | Primary brand candidate |
| `move-tree.svg` | Board square feeding a move tree | Developer docs and engine workflows |
| `fen-grid.svg` | FEN-like text grid becoming a workflow | CLI and deterministic output emphasis |
| `rtk-monogram.svg` | Compact RTK board-tile monogram | Favicon, app icon, release avatar |
| `engine-line.svg` | Board tile plus evaluation/PV line | Engine analysis and dataset workflows |

Open `preview.html` in a browser to compare all five together.

The `non-board/` directory contains a third round that avoids board imagery and
focuses on CLI, engine, data, publishing, and reproducibility marks.

The `signature/` directory contains a fourth round of more deliberate identity
candidates: state lattice, FEN helix, proof loop, kernel orbit, command sigil,
record weave, transposition knot, and RTK crest marks.
